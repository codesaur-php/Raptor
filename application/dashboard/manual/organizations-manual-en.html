What are Organizations?

Organizations is a module for managing system organizations.

  • Hierarchical structure - Organize organizations in a hierarchy
  • RBAC alias system - Assign RBAC aliases to organizations as permission groups
  • Logo management - Manage organization logo images

RBAC Alias

Each organization has an RBAC alias. Permissions for the organization's members are configured under this alias. (See: RBAC Manual)

For example: an organization with the alias university will have its users' permissions managed under the university group.

An organization with the common alias will have its users' permissions managed under the common group. This is useful for applying shared permission settings across multiple organizations.
