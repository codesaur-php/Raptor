What are Files?

Files is a module to view all uploaded files across all modules in one place on the Dashboard. It displays all files including images, videos, audio, PDFs, and documents in a table format.

Table Selection

At the top of the page, a list of file tables is displayed as pill badges. Each table shows the file count and total size.

  • files - Main file table. Full upload, edit, and delete capabilities.
  • attachments - Attachment files from other modules (news, pages, products, etc.). View-only - adding and removing files is done from the respective module's edit page.

File Preview

  • View images enlarged on an overlay
  • Play video/audio directly
  • Open PDF files in an iframe
  • Fetch and display text files

File Information

Each file has the following modals:

  • Location - The file's server path
  • Code tag - HTML embed code for inserting into content

Statistics

The following statistics are displayed at the top of the page:

  • Total number of tables
  • Number of files
  • Total size

File Upload (files table only)

File upload is available when the files table is selected.

  • Select files - Multiple files can be selected at once
  • Optimize images - Automatically optimizes image files for web
  • Allowed file types: images, video, audio, documents, archives
  • File size limit depends on server configuration

Edit / Delete (files table only)

  • Edit - Modify file description, keywords (system_content_update permission required)
  • Delete - Permanently deletes the file record from the database; a backup copy is stored in the Trash table for recovery. The actual file is not removed from storage (system_content_delete permission required)
Attachment table files can only be edited or removed from their respective module's (News, Pages, Products) edit page.
