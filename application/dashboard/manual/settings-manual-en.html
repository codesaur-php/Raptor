What are Settings?

Settings is a module for managing general website configuration. It has three tabs: Meta, Logo, Config.

Meta Tab

Multi-language fields:

FieldDescription
TitleWebsite title, up to 70 characters
DescriptionWebsite description, up to 255 characters
Urgent NoticeUrgent notification, supports HTML
ContactContact information, supports HTML
AddressAddress displayed on the website, supports HTML
CopyrightCopyright notice

Global fields (language-independent):

  • Email - Email address displayed on the website
  • Phone - Phone number displayed on the website

Logo Tab

FieldDescription
LogoLogo image per language (select, replace, delete)
Favicon.ico file
Apple Touch Icon180x180 pixel image

Image preview and file size are displayed.

Config Tab

Additional configuration in JSON format.

  • Pretty Print button to format JSON
  • Examples: google_analytics, maintenance_mode, etc.

Saving

Each tab has its own save button. Data is saved via AJAX fetch.

Log

If you have the system_logger permission, settings-* logs are displayed.
