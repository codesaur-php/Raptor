Захиалга гэж юу вэ?

Orders нь вэбсайтаас ирсэн бүтээгдэхүүний захиалгыг удирдах модуль юм.

Захиалгын модуль нь дараах онцлогуудтай:

  • Захиалгын статус workflow - Шинэ -> Боловсруулж байна -> Баталгаажсан -> Илгээсэн -> Дууссан
  • И-мэйл мэдэгдэл - Хэрэглэгчид автомат мэдэгдэл илгээнэ
  • Хэрэглэгчийн мэдээлэл - Захиалагчийн нэр, и-мэйл хаяг

Захиалгын жагсаалт

Dashboard цэснээс Захиалгууд дарахад жагсаалтын хуудас нээгдэнэ.

Шүүлтүүр

Дараах шүүлтүүрээр захиалгуудыг ялгаж харж болно:

  • Хэл - mn, en гэх мэт хэлний кодоор
  • Статус - Захиалгын төлвөөр (шинэ, боловсруулж байна, баталгаажсан гэх мэт)

Захиалга харах

Жагсаалтаас харах (нүд) товчийг дарна.

  • Захиалгын дэлгэрэнгүй мэдээлэл - бүтээгдэхүүн, тоо хэмжээ, хэрэглэгчийн мэдээлэл
  • Статус өөрчлөх - dropdown-оос шинэ статус сонгоно
  • Хэрэглэгчид мэйл илгээх боломжтой

Мэдэгдэл (Discord + И-мэйл)

Discord мэдэгдэл

Захиалгын статус өөрчлөгдөх үед Discord webhook-ээр автомат мэдэгдэл илгээгдэнэ (webhook тохируулсан бол).

ҮйлдэлМэдэгдэлд орох мэдээлэл
Статус өөрчлөхЗахиалгын ID, хэрэглэгчийн нэр, хуучин статус, шинэ статус, хэн өөрчилсөн
Админ имэйл мэдэгдэл

Шинэ захиалга ирэх бүрд админд автомат имэйл мэдэгдэл илгээх боломжтой. Энэ тохиргоо нь жагсаалтын хуудасны дээд хэсэгт байрлана (system_coder эрхтэй хэрэглэгчдэд харагдана).

ТохиргооТайлбар
Идэвхжүүлэх / Идэвхгүй болгох Toggle switch ON = имэйл хаяг тохируулна, OFF = имэйл хаягийг хоослоно
Энэ тохиргоо нь .env файлын RAPTOR_ORDER_EMAIL_TO утгыг шууд өөрчилнө. Имэйл хаяг тохируулсан = идэвхтэй, хоосон = идэвхгүй.
Хэрэглэгчийн имэйл мэдэгдэл

Захиалгатай холбоотой дараах үед хэрэглэгчид автомат и-мэйл илгээгдэнэ:

ҮйлдэлХүлээн авагчЗагварАгуулга
Шинэ захиалга (вэбсайтаас) Захиалагч order-confirmation Захиалгын ID, хэрэглэгчийн нэр, бүтээгдэхүүн, тоо хэмжээ, JWT баталгаажуулах линк
Статус өөрчлөх (Dashboard-аас) Захиалагч order-status-update Захиалгын ID, хуучин/шинэ статус, хэрэглэгчийн нэр, бүтээгдэхүүн

Захиалга устгах

Жагсаалтаас устгах (хогийн сав) товчийг дарна.

  • Баталгаажуулах цонх гарна (SweetAlert2)
  • Мэдээллийн сангаас бүрмөсөн устгагдана; нөөц хуулбар нь Trash хүснэгтэд хадгалагдана
  • system_product_delete эрх шаардлагатай
