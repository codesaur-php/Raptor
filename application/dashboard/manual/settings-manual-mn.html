Тохиргоо гэж юу вэ?

Settings нь вэбсайтын ерөнхий тохиргоог удирдах модуль юм. Гурван табтай: Meta, Logo, Config.

Meta таб

Олон хэл дээрх талбарууд:

ТалбарТайлбар
Гарчиг (title)Вэбсайтын гарчиг, 70 тэмдэгт хүртэл
Тайлбар (description)Вэбсайтын тайлбар, 255 тэмдэгт хүртэл
Яаралтай мэдэгдэл (urgent)Яаралтай мэдэгдэл, HTML дэмжинэ
Холбоо барих (contact)Холбоо барих мэдээлэл, HTML дэмжинэ
Хаяг (address)Вэбсайт дээр харагдах хаяг, HTML дэмжинэ
CopyrightЗохиогчийн эрхийн мэдэгдэл

Глобал талбарууд (хэлнээс үл хамаарах):

  • И-мэйл - Вэбсайт дээр харагдах и-мэйл хаяг
  • Утас - Вэбсайт дээр харагдах утасны дугаар

Config таб

JSON хэлбэртэй нэмэлт тохиргоо.

  • Pretty Print товчоор JSON-г форматлах
  • Жишээ: google_analytics, maintenance_mode гэх мэт

Хадгалах

Таб бүрт тусдаа хадгалах товч байна. AJAX fetch-ээр хадгалагдана.

Лог

system_logger эрхтэй бол settings-* лог харагдана.
