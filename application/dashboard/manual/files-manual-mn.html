Файлууд гэж юу вэ?

Files нь Dashboard дээр бүх модулиудаас байршуулсан файлуудыг нэг дороос харах модуль юм. Зураг, видео, аудио, PDF, баримт бичиг зэрэг бүх файлуудыг хүснэгтээр харуулна.

Хүснэгт сонгох

Хуудасны дээд хэсэгт файлын хүснэгтүүдийн жагсаалт (pill badge) харагдана. Хүснэгт бүрт файлын тоо, нийт хэмжээ харуулна.

  • files - Үндсэн файлын хүснэгт. Файл байршуулах, засварлах, устгах бүрэн боломжтой.
  • attachments - Бусад модулиудын хавсралт файлууд (news, pages, products г.м.). Зөвхөн жагсаалт харах боломжтой - файл нэмэх, устгах нь тухайн модулийн засварлах хуудаснаас хийгдэнэ.

Файлын урьдчилан харах

  • Зургийг overlay дээр томруулж харах
  • Видео/аудио шууд тоглуулах
  • PDF файлыг iframe-д нээх
  • Текст файлыг fetch хийж харуулах

Файлын мэдээлэл

Файл бүрт дараах modal-ууд бий:

  • Location - Файлын серверийн зам (байршил)
  • Code tag - HTML-д оруулах embed код (HTML tag)

Статистик

Хуудасны дээд хэсэгт дараах статистик мэдээлэл харагдана:

  • Нийт хүснэгтийн тоо
  • Файлын тоо
  • Нийт хэмжээ

Файл байршуулах (зөвхөн files хүснэгт)

files хүснэгт сонгогдсон үед файл байршуулах боломжтой.

  • Файл сонгох - Олон файл зэрэг сонгож болно
  • Зураг optimize - Зургийн файлыг автоматаар web-д зориулж optimize хийнэ
  • Зөвшөөрөгдөх файлын төрөл: зураг, видео, аудио, баримт бичиг, архив
  • Файлын хэмжээний хязгаарлалт серверийн тохиргооноос хамаарна

Засварлах / Устгах (зөвхөн files хүснэгт)

  • Засварлах - Файлын тайлбар, түлхүүр үг зэргийг өөрчлөх (system_content_update эрх шаардлагатай)
  • Устгах - Файлын бичлэгийг мэдээллийн сангаас бүрмөсөн устгана; нөөц хуулбар нь Trash хүснэгтэд хадгалагдана. Бодит файл санах ойгоос устахгүй (system_content_delete эрх шаардлагатай)
Attachments хүснэгтүүдийн файлуудыг засварлах, устгах нь тухайн модулийн (News, Pages, Products) засварлах хуудаснаас хийгдэнэ.
