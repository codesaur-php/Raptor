Файлууд гэж юу вэ?

Files нь Dashboard дээр бүх модулиудаас байршуулсан файлуудыг нэг дороос харах модуль юм. Зураг, видео, аудио, PDF, баримт бичиг зэрэг бүх файлуудыг хүснэгтээр харуулна.

Хүснэгт сонгох

Хуудасны дээд хэсэгт файлын хүснэгтүүдийн жагсаалт (pill badge) харагдана. Хүснэгт бүрт файлын тоо, нийт хэмжээ харуулна.

  • files бол үндсэн хүснэгт
  • Бусад нь attachments хэлбэртэй

Файлын урьдчилан харах

  • Зургийг overlay дээр томруулж харах
  • Видео/аудио шууд тоглуулах
  • PDF файлыг iframe-д нээх
  • Текст файлыг fetch хийж харуулах

Файлын мэдээлэл

Файл бүрт дараах modal-ууд бий:

  • Location - Файлын серверийн зам (байршил)
  • Code tag - HTML-д оруулах embed код (HTML tag)

Статистик

Хуудасны дээд хэсэгт дараах статистик мэдээлэл харагдана:

  • Нийт хүснэгтийн тоо
  • Файлын тоо
  • Нийт хэмжээ
