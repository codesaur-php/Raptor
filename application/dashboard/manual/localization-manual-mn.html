Нутагшуулалт гэж юу вэ?

Localization нь системийн хэл болон орчуулгын текстүүдийг удирдах модуль юм. Хоёр табтай: Хэл (Language) ба Текст (Localization).

Хэлний удирдлага

Хэлний хүснэгтэд дараах мэдээлэл харагдана:

БаганаТайлбар
КодХэлний код (жишээ нь: mn, en)
ТугХэлний тугны зураг
LocaleLocale тохиргоо
НэрХэлний нэр
ТайлбарНэмэлт тайлбар
ОгнооҮүсгэсэн огноо

CRUD үйлдлүүд (үүсгэх, унших, засварлах, устгах) хийх боломжтой.

Шинэ хэл нэмэхэд автоматаар бүх localized контентыг эх хэлнээс хуулна (*_content хүснэгтүүд). Default хэлийг устгах боломжгүй.

Орчуулгын текст

Системийн UI текстүүд (keyword -> хэл бүрт орчуулга) энэ табд удирдагдана.

БаганаТайлбар
Түлхүүр үгОрчуулгын keyword (жишээ нь: home, login)
Хэл бүрийн текстХэл тус бүрийн орчуулга
Төрөлplain эсвэл html

CRUD үйлдлүүд (үүсгэх, унших, засварлах, устгах) хийх боломжтой.

Лог

system_logger эрхтэй бол хуудасны доод хэсэгт сүүлийн үйлдлүүдийн лог харагдана.
