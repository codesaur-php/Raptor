What is a Product?

Products is the product management module of Raptor Dashboard. All operations such as creating, editing, deleting, and publishing products are performed through this module.

Products have the following features:

  • Multi-language - Create separate products for each language
  • SEO-friendly slug - URL slug is automatically generated from the title
  • Type & Category - Classify products by type and category
  • Price information - Set product pricing
  • File attachments - Attach images and other files

Product List

Click Products from the Dashboard menu to open the list page.

Filters

You can filter products by the following criteria:

  • Language - By language code such as mn, en
  • Type - By product type
  • Category - By product category
  • Status - Published or draft

HTML Content Validation
Content with broken HTML structure will not be saved. If unclosed HTML comments (<!-- ... ) or broken tags cause significant content loss, an error message is displayed. Fix the HTML in Source mode and try again.
Publish Settings

Users with system_content_publish permission will see the following options:

  • Publish - Makes the product visible on the website
  • Featured - Includes the product in featured products on the homepage
Automatic Actions
  • Slug - Automatically generated from the title (Mongolian Cyrillic to Latin transliteration)
  • Description - If empty, an excerpt is generated from the content

Edit Product

Click the edit (pencil) button from the list.

  • All fields can be modified (except slug)
  • File attachment management - add new files, edit descriptions, delete
  • Editing a published product requires system_content_publish permission
  • Content with broken HTML structure will not be saved (error message is displayed)
  • If no changes are made, saving is skipped ("No update!" error)
Change Log

Users with system_logger permission will see the last 10 action logs on the edit page. Clicking a log entry opens detailed information in a modal.

View Product

Click the view (eye) button from the list.

  • All fields are displayed in read-only mode
  • Content is displayed in moedit read-only mode (moedit preview)
  • View header image and attached files
  • Publication status and date

Delete Product

Click the delete (trash) button from the list.

  • A confirmation dialog appears (SweetAlert2)
  • Permanently deleted from the database; a backup copy is stored in the Trash table for recovery
  • system_product_delete permission is required

Discord Notification

Sample Data

Products with category='_raptor_sample_' are sample data. When sample data is detected, a yellow warning and a Reset & Production button appear on the list page.

Warning: Resetting will only delete sample data and their attachment records. User-created data will remain intact. This action cannot be undone.
Identifying sample data: If you want to use some as real content, simply change the product's category (e.g., to general) and save - those will not be deleted during a reset.
