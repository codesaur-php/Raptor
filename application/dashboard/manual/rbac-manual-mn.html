RBAC (Role-Based Access Control) гэж юу вэ?

Хэрэглэгчдийн эрхийн удирдлагын модуль. Дүр (Role) болон Эрх (Permission) хоёрын хослолоор хэрэглэгч бүрийн хандах эрхийг тодорхойлно. Байгууллага бүрт тусдаа RBAC alias-ээр эрхийн бүлэг үүсгэнэ.

  • Alias систем - Байгууллага бүрт тусдаа alias (жишээ: system, common, university)
  • Дүр (Role) - Хэрэглэгчид оноогдох дүр (admin, manager, editor, viewer гэх мэт)
  • Эрх (Permission) - Тодорхой үйлдэл хийх зөвшөөрөл (user_insert, content_delete гэх мэт)
  • Матриц харагдац - Дүр x Эрх хүснэгтээр нүд бүрт checkbox-оор удирдана

Матриц хүснэгт

RBAC-ийн үндсэн хуудас нь дүр (мөр) x эрх (багана) хүснэгт хэлбэрээр харагдана.

  • Checkbox тэмдэглэхэд тухайн дүрт эрх оноогдоно (AJAX POST)
  • Checkbox арилгахад эрх хасагдана (AJAX DELETE)
  • Эрхүүд модулиар бүлэглэгдсэн (user, content, product, localization гэх мэт)
  • Баруун дээд талд Add Role, Add Permission товчнууд

Alias

Байгууллага бүрт тусдаа RBAC alias бий. Alias-ийн дор дүрүүд ба эрхүүд бүлэглэгдэнэ.

  • system - Системийн үндсэн alias
  • common - Олон байгууллагад нийтлэг alias, common alias-тай байгууллагуудын хэрэглэгчдийн эрх нийтлэг удирдагдана
  • Байгууллагын alias (жишээ: university) - Тусдаа эрхийн бүлэг
  • Байгууллагын жагсаалтаас alias дээр дарахад RBAC матриц руу шилжинэ (Дэлгэрэнгүй: Байгууллага manual)

Шинэ дүр нэмэх

Modal хэлбэрээр. Талбарууд:

  • Нэр (name) - заавал, давтагдахгүй, 128 тэмдэгт хүртэл
  • Тайлбар (description) - заавал биш
system alias-д coder нэртэй дүр үүсгэх боломжгүй (системд нөөцлөгдсөн).

Шинэ эрх нэмэх

Modal хэлбэрээр. Талбарууд:

  • Модуль (module) - заавал биш, байгаа модулиудаас сонгох эсвэл шинэ бичих (анхдагч: general)
  • Нэр (name) - заавал, давтагдахгүй, 128 тэмдэгт
  • Тайлбар (description) - заавал биш
system_coder нэртэй эрх үүсгэх боломжгүй (нөөцлөгдсөн).

Эрх шалгах механизм

Хэрэглэгч нэвтрэхэд RBAC системд бүх дүрүүд болон тэдгээрийн эрхүүдийг санах ойд ачаална. Эрх шалгах key формат: {alias}_{name} (жишээ: system_rbac, system_content_delete).

  • system_coder дүртэй хэрэглэгч бүх эрхтэй (шалгалт хийхгүй)

Лог

system_logger эрхтэй бол хуудасны доод хэсэгт RBAC-тай холбоотой сүүлийн үйлдлүүдийн лог харагдана.
