What is Localization?

Localization is a module for managing system languages and translation texts. It has two tabs: Language and Localization (Text).

Language Management

The language table displays the following information:

ColumnDescription
CodeLanguage code (e.g., mn, en)
FlagLanguage flag image
LocaleLocale setting
NameLanguage name
DescriptionAdditional description
DateCreation date

CRUD operations (create, read, update, delete) are available.

When adding a new language, all localized content is automatically copied from the source language (*_content tables). The default language cannot be deleted.

Translation Text

System UI texts (keyword -> translation per language) are managed in this tab.

ColumnDescription
KeywordTranslation keyword (e.g., home, login)
Language textsTranslation for each language
Typeplain or html

CRUD operations (create, read, update, delete) are available.

Log

If you have the system_logger permission, the recent activity log is displayed at the bottom of the page.
