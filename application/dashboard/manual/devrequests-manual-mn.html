Хөгжүүлэлтийн хүсэлт гэж юу вэ?

Dev Requests нь програмистуудад хүсэлт, алдааны мэдэгдэл илгээх модуль юм.

  • Хүсэлт-хариулт thread бүтэц - Хүсэлт бүрт хариултын thread үүснэ
  • Файл хавсралт - Хүсэлт болон хариултад файл хавсаргах
  • И-мэйл + Discord мэдэгдэл - Автомат мэдэгдэл илгээх
  • Статус удирдлага - Хүсэлтийн явцыг хянах

Discord мэдэгдэл

Webhook тохируулсан бол дараах үед Discord мэдэгдэл илгээгдэнэ:

И-мэйл мэдэгдэл
ҮйлдэлХүлээн авагчЗагварАгуулга

Хариулт бичих Илгээгч эсвэл сүүлийн хариулагч * dev-request-response Хүсэлтийн ID, гарчиг, хариулагчийн нэр, хариултын агуулга, линк

* Илгээгч хариулбал - сүүлийн хариулагч (програмист) мэдэгдэл авна. Програмист хариулбал - эх илгээгч мэдэгдэл авна.
