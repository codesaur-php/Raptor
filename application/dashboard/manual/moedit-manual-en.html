What is moedit?

moedit is a WYSIWYG text editor used in the Raptor Dashboard. It is designed for preparing content such as pages, news, and products.

Used in the following Dashboard sections:

  • Pages - Create, edit, and view pages
  • News - Create, edit, and view news articles
  • Products - Create, edit, and view products

When the content type is embed, the editor automatically switches to HTML Source mode. Otherwise, it operates in Visual mode.

Fullscreen Mode

Click the button in the toolbar to enter fullscreen mode.

  • The content title is displayed at the top
  • Press Escape or click the button again to exit
  • Provides more space for editing content

Paste Handling

Paste TypeDescription
HTML content HTML is automatically cleaned while preserving semantic structure
Plain text Each line is converted to a <p> tag
Image (Clipboard) Clipboard images are automatically detected and uploaded to the server
Word document Word-specific formatting is cleaned. Warning shown if local images are present (cannot be transferred)

Image Optimization

When the Optimize Images checkbox at the beginning of the toolbar is checked, images are automatically compressed during upload (50-80% size reduction while preserving quality).

Applies to:

  • Header image
  • Images inserted into content
  • Image-type file attachments

Header Image

Used for the featured/cover image of news articles, products, etc. Only one header image can be set per content.

  • Click the button in the toolbar
  • After uploading, the image appears at the top of the editor
  • To change - click the (Change) button on hover
  • To remove - click the (Remove) button on hover

Insert Image

Click the button in the toolbar. A dialog opens with two tabs. You can insert as many images as needed into the content.

TabDescription
From ComputerSelect an image file from your computer
From URLEnter an image URL - real-time preview is shown
Tips:
  • You can drag and drop images directly into the editor
  • You can paste images from the clipboard (Ctrl+V) - screenshots work too
  • Uploaded images are automatically saved to the server
  • Image sizing: max-width: 100%, height: auto

File Attachments

Click the button in the toolbar to attach files to the content. You can attach as many files as needed.

The attachment table displays:

  • File name - with download link
  • Properties - file type (extension) and size
  • Description - editable text field
  • Date - upload date
  • Delete - remove individual attachments

In Update mode, previously attached files are automatically loaded.

Insert Link / Email

  1. Select the text you want to link
  2. Click the button in the toolbar
  3. Choose URL or Email mode
ModeDescription
URL Enter a link. https:// is auto-added. Opens in a new tab (target="_blank")
Email Enter an email address. mailto: is auto-added

The Display text field is optional - if left empty, the selected text is used.

Insert YouTube Video

  1. Click the button in the toolbar
  2. Enter the YouTube embed code or URL
Recommendation: On YouTube, click Share -> Embed and copy the <iframe> code for best results. Pasting a URL will use estimated dimensions.

Supported URL formats:

  • https://www.youtube.com/watch?v=VIDEO_ID
  • https://youtu.be/VIDEO_ID

AI OCR - Recognize Text from Images

  1. Insert an image containing text into the content
  2. Click the button in the toolbar
  3. Optionally enter AI instructions (can be left empty)
  4. Click Convert

AI recognizes text in the image and automatically converts it to HTML:

  • Tables -> <table>
  • Lists -> <ul> / <ol>
  • Headings -> <h1>-<h6>
  • Text -> <p>

AI PDF -> HTML

  1. Click the button in the toolbar
  2. Select a PDF file
  3. Choose pages if the PDF has multiple pages
  4. Optionally enter AI instructions
  5. Click Insert

AI recognizes the PDF structure (tables, lists, headings, text) and converts it to HTML within the content.

AI Shine - Beautify Content

  1. Prepare your content (plain text, lists, etc.)
  2. Click the button in the toolbar
  3. Optionally enter AI instructions
  4. Click Apply

AI analyzes the content structure and beautifies it using Bootstrap 5 components:

Content TypeTransformed To
List informationCard or list-group
Comparisons / Multi-columnTable (table-striped, table-hover)
Q&A / FAQAccordion
Step-by-stepList-group or card
Important informationAlert or callout
Related itemsRow/col grid

Text content is preserved - only the HTML structure is beautified.

Edit HTML Source

Click the button in the toolbar to toggle between Visual and Source mode.

  • In Source mode, you can directly edit the HTML code
  • Code is auto-formatted with 2-space indentation
  • Cursor position is preserved between modes
  • Press Ctrl+S to switch back to Visual mode
When the content type is embed, Source mode is automatically activated.

Readonly Mode

When content is opened in View mode, the editor operates in read-only mode:

  • All editing tools are hidden
  • Only Print, Source (read-only), and Fullscreen buttons remain
  • Content cannot be edited
