What are Dev Requests?

Dev Requests is a module for sending requests and bug reports to developers.

  • Request-reply thread structure - Each request has a reply thread
  • File attachments - Attach files to requests and replies
  • Email + Discord notifications - Automatic notification delivery
  • Status management - Track request progress

Regular users can only see their own requests and requests assigned to them. Users with system_development permission can see all requests.

Notifications (Discord + Email)

Discord Notification

When a webhook is configured, Discord notifications are sent for:

ActionNotification details
New request createdRequest ID, title, author name, assigned person
Reply postedRequest ID, title, responder name, new status
Email Notification
ActionRecipientTemplateContent
New request Assigned person (if any) dev-request-new Request ID, title, author name, link to request
Reply posted Creator or last responder * dev-request-response Request ID, title, responder name, response content, link

* If the creator replies - the last responder (developer) receives the notification. If a developer replies - the original creator receives the notification.

Delete

  • Permanently deleted; a backup copy is stored in the Trash table
